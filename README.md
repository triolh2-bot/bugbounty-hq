# 🔓 BugBountyHQ

Professional bug bounty program management platform.

## Features

- Program Management (BBP, VDP)
- Submission Tracking
- Researcher Portal
- Bounty Calculation
- API Access
- Webhook Integration (HackerOne, Bugcrowd)

## Installation

Supported runtime: Python 3.13.12. The repo includes `.python-version` for local
tooling and uses `requirements.txt` as the pinned constraints file.

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e . -c requirements.txt
python app.py
```

Access at http://localhost:5000
